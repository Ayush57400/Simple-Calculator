# Calculator App

A simple and responsive calculator built using **HTML, CSS, and JavaScript**.

## Features
- Addition, subtraction, multiplication, division
- Percentage support
- Decimal calculations
- AC (clear) button
- DEL (delete last character) button
- Keyboard support
- Clean modern UI

## Technologies Used
- HTML
- CSS
- JavaScript
