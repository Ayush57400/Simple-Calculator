# Simple Calculator

A modern, responsive calculator web app built using **HTML, CSS, and JavaScript**.  
This project supports standard arithmetic operations, keyboard input, theme switching, and calculation history.

---

## Features

- Basic arithmetic operations
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Percentage
- Responsive modern UI
- Dark / Light mode toggle
- Calculation history with local storage
- Keyboard support
- Clear and delete functions
- Mobile-friendly layout

---

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
